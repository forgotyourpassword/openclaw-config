# Home Dashboard

A simple control panel for house projects and decisions.

## Active projects
- Add links here as projects start

## Waiting on
- Quotes
- Measurements
- Scheduling

## Quick capture
When something comes up, choose one:
- New actionable project → create a note from `Templates/Project Template.md` in `Projects/Active/`
- House knowledge / issue / measurements → add it to an `Areas/` note
- Vendor / contractor / quote source → create a note from `Templates/Contractor Template.md`
- Purchase worth remembering → create a note from `Templates/Purchase Template.md`
- Big decision → append to `Decisions.md`

## Review rhythm
### Weekly
- Review active projects
- Move blocked items into `Waiting on`
- Update next actions
- Archive done projects into `Projects/Done/`

### After work happens
Capture immediately:
- what changed
- who did it
- cost
- model/material details
- photos/receipts links

## Principles
- Keep notes useful, not perfect
- One note per real project
- House knowledge lives in `Areas/`
- Decisions go in `Decisions.md`
- Don’t build a giant system you won’t maintain
